# ADN DENTAL - Deployment Package

This package contains the complete source code for the ADN DENTAL website. The site is built with a premium, high-performance design system suitable for a world-class dental equipment supplier.

## 📁 File Structure

To host this website, upload the following files to your server's root directory (usually `public_html` or `www`):

### Core Files
- `index.html` - The homepage.
- `style.css` - The global design system and animations.
- `data.js` - Multi-language support (English/French) and interactive logic.

### Category & Partner Pages
- `catalog.html` - Premium partner selection (DGSHAPE / ZUBLER).
- `dgshape.html` - DGSHAPE product showcase.
- `zubler.html` - ZUBLER product showcase.

### Product Detail Pages
- `product-dwx-53dc.html`
- `product-dwx-52d-plus.html`
- `product-dwx-42w.html`
- `product-dwx-52dci.html`
- `product-vario-press-300e.html`
- `product-v60-iplus.html`
- `product-vario-200zr.html`
- `product-z1-eco.html`
- `product-z1-cam.html`

### Information Pages
- `about.html` - Company story and statistics.
- `news.html` - Latest updates and partnerships.
- `courses.html` - Training and workshops.
- `support.html` - Technical support request form.
- `contact.html` - Contact details and location map.

## 🚀 How to Host

### Option A: Free & Fast (Recommended)
1. Go to [Vercel](https://vercel.com) or [Netlify](https://netlify.com).
2. Drag and drop this folder into their dashboard.
3. Your site will be live in seconds with a free SSL certificate.

### Option B: Traditional Hosting (.dz Domain)
1. Purchase a hosting plan from an Algerian provider (e.g., Icosnet).
2. Use an FTP client like **FileZilla**.
3. Upload all the files listed above to the `public_html` folder.

## 🛠️ Customization
- **Images:** Replace the `picsum.photos` URLs in the HTML files with your actual product photography.
- **Contact Info:** Update the phone numbers and email in `contact.html` and `data.js`.
- **WhatsApp:** Update the WhatsApp link in `index.html` and product pages with your official number.
